# installation

to install:

``` bash
$ pip install metrics.gitinfo
```

to uninstall:

``` bash
$ pip uninstall metrics.gitinfo
```

for details please see the documentation of metrics.

# metrics.gitinfo

The **metrics.gitinfo** package is a plugin for the existing metrics package.
